# Patients Monitor
This project is a simulation of patients monitoring system.
The stack includes react.js, redux.js, redux-observable.js, among other technologies.

Requirements:

- Node.js v8.9.4
- Yarn v1.10.1


Instructions:

1) Clone repo in your machine
2) run 'yarn install'
3) run 'yarn json-server'
4) run 'yarn start'
